Windows Object:

Variable that are declared without the words “var, let, const” are global variable and can be called using window.[variable_name].

The window.location property is an object that contains information about the URL of the current page, here is a list of property you can use:

  • Href (Returns the full URL)
  • Protocol (Returns a string with the protocol such as hht, https, etc…)
  • Host (Returns a sting with the domain)
  • Hostname (Returns a string with the domain URL)
  • Port (Returns a string with the port number)
  • Pathname (Returns a string of the path that follows the domain)
  • Search (Returns a string with the query parameters)
  • Hash (Returns a string that starts with a “#” followed by the fragment identifier)
  • Origin (Returns a string that shows the protocol and domain where the current page originated from)
