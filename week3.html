First thing, Object work by reference, that meaning that if we assign an object to a value, and have the same object to another value, changing information to just one value, will change the object for both of the values equally.
The “This” keyword, that the definition says “The keyword this refers to the object that it is within. It can be used inside methods to gain access to the object’s properties.”
