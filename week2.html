Most of the Things from this week where actually familiar, I already had some knowledge of Loops, Logic and Functions coming from other experience and other classes, so it was nice to read and remember some little things.

One topic from this week that is actually always been a struggle for me, is Arrays. I worked a bit with arrays but I have always done my best to avoid it, I prefer hard mathematical problems to solve then having to do with arrays. But I did my best this week to gain a greater knowledge of Arrays and did my absolute best to get to know important functions like:

  • Delete (deletes an element from the array according to the index given)
  • Length (gives us the length of the array)
  • Pop (deletes the last element of the array)
  • Shift (deletes the first item in the array)
  • Push (appends a new value to the end of the array)
  • Unshift (appends a new value to the start of the array)
  • Contact (merges different arrays)
  • Join (Turns the array in a string)
  • Slicing (create a subarray slicing the existing array with the index parameter given)
  • Splicing (Substitute the index given to the new value given)
  • Sort (alphabetically orders the array)
  • indexOf (gives us the index of the value in the Array)
