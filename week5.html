The first thing is “Strict mode”, this was introduced in ES5, and what it does is that stopes you from running with errors, which it is what is usually happens in JS, so that if you have a variable, that is not define instead of using a Warning it will give you an exception. Strict mode is invoked by using the line: 'use strict';
